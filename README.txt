Ultimate Post Generator is a WordPress plugin that integrates with OpenAI's powerful GPT models to generate rich and engaging content.

== Description ==

The Ultimate Post Generator plugin harnesses the power of OpenAI's GPT-3.5 to create dynamic, interesting, and contextually relevant content for your WordPress site. It's designed to streamline content creation, from blog posts to articles, using advanced AI technology.

Key Features:
- OpenAI GPT-3.5 Integration: Connects seamlessly with OpenAI to generate quality content.
- Custom Content Formatting: Automatically formats AI-generated content with HTML tags.
- User-Friendly Interface: Intuitive settings and operation within the WordPress admin panel.
- AJAX Real-Time Interaction: Provides a smooth, responsive user experience without page reloads.
- Draft Post Creation: Easily convert AI-generated content into WordPress draft posts.
- SEO-Friendly: Includes functionality to add SEO tags to generated content.
- Secure and Reliable: Implements WordPress best practices for security and data sanitization.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ultimate-post-generator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Ultimate Post Generator screen to configure the plugin.
4. Enter your OpenAI API key and set your preferred maximum tokens.

== Frequently Asked Questions ==

= How do I get an OpenAI API key? =
You can obtain an API key by signing up at OpenAI's website and registering for API access.

= Is this plugin free to use? =
Yes, the Ultimate Post Generator is free, but using the OpenAI API may incur costs based on their pricing model.
